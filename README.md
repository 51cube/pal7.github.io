第二十五章
天神慑魄隐心机
月清疏一行人艰难自照胆返回，虽以言语暂时欺骗过敖胥，但还是心有余悸。只好先前往天魔国暂时休整一番，再做打算……
众人经炎波返回魔界，不料修吾突然凶性大发，众人方才明白，敖胥方才假意给修吾治伤，实则施下咒毒。月清疏利用共生联系，削弱修吾，终于将其制服，并将其带至天魔国。

第二十六章
惊觉枝蔓叶障目
桑游带领众人返回至毒瘴泉中，向卫戍禀明一切。卫戍应允，但保险起见，还是让桑游等人到结界附近巡查一番……
桑游带领众人返回毒瘴泉中，请求卫戍照看子秋，并以毒瘴为修吾恢复灵力。众人认识到敖胥的真面目，但始终无法窥得其目的。天师门与敖胥素有来往，而白家有与孟章有所关联，于是众人决定分头行动，调查白家与天师门。

第二十七章
道门狼子露端倪
调查天师门已然势在必行，不过听闻各地凶兽再度肆虐，还是先到萍溪村看看村民们有无危险……
修吾与月清疏来至天师门，却发现天师门空空荡荡。后二人联合三长老，利用修吾的上神身份，诱使孟章说出敖胥计划。孟章见计划败露冲出天师门。以凶兽内丹之力力战修吾，但终究不是对手，而后施展空间法术遁走。

第二十八章
本是同根手足情
白茉晴与桑游悄悄来至卢龙府，欲调查白家在敖胥计划中所扮演的角色。但白茉晴放心不下当日流民的情况，决定先到粥铺去看看……
白茉晴与桑游来至卢龙府，二人偷偷潜入白府。白茉晴知晓密室所在，遂前往调查。方知白家挖掘冰晶石，正是为敖胥大阵提供灵力。二人更是发现了龙袍与玉玺，白茉晴方知兄长有谋逆之心。

第二十九章
真相既白愤难平
月清疏与修吾调查清楚孟章的阴谋，得知子秋乃是敖胥计划中的重要一环，遂来至泉隐村。二人决定先找桑湄商量对策……
月清疏一行人于泉隐村汇合。众人将线索联系起来，但依旧无法洞悉敖胥的计划。恰逢此时，毒瘴器偶与凶兽进攻泉隐村，并将子秋夺走。众人一番苦战，将凶兽九婴自咒毒中解放，方知凶兽乃敖胥释放至人界。

第三十章
雏燕身如不系舟
子秋再度落入敖胥手中，但子秋留下讯息说若是自己被抓走可至天魔国寻求葵羽帮助。众人无法，只得再度前往天魔国……
月清疏一行人来至天魔国，葵羽将春滋剑还给修吾。余霞真人虽知晓了破阵之法，但由于器偶存在，神庭阵可生生不息。器偶核心乃灵兽元灵，明庶门又精于御灵一道，月清疏决定返回明庶门，寻找破除器偶之法。

第三十一章
共把衷肠慰新愁
器偶乃是凭借灵兽元灵方能生生不息，月清疏认为明庶门与灵兽渊源颇深，或许可以在门派中寻得破解器偶之法……
月清疏寻得破解器偶之法。大战在即，月清疏心绪不宁，修吾寻得月清疏，二人互诉衷肠。白茉晴因为自己兄长缘故，自觉无法面对桑游，尽管桑游宽慰自己，但她始终无法释怀。

第三十二章
提携玉龙神庭破
月清疏一行人跟随仙盟弟子来至神庭塔营地附近，众人等待余霞真人发号施令……
修吾以春滋剑破开结界，孟章操控六器偶前来阻挡，被月清疏以明庶门术法破除。众人赶至塔顶，与被敖胥附身的子秋战斗。子秋凭葵羽灵力，使神智暂时恢复，以反制之法将敖胥赶出体内。但神庭阵却并未停下……

第三十三章
斩木逆天归去留
虽然阻止了孟章，救回子秋，但大阵并未停下来。余霞真人推测出此阵为天地双阵，另有一阵应当在神界。修吾猜测另一处阵法应当在春滋，于是，月清疏与修吾决定前往春滋，与敖胥决战……
